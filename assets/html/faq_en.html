📤 Upload & Files
What file formats are supported?

Audio: MP3, WAV, FLAC, M4A, AAC, OGG, WMA

Video: MP4, MOV, AVI, MKV, WEBM

When uploading a video file, the system will automatically extract the audio track for separation.

💡 For best results, use lossless formats like WAV or FLAC.
Is there a file size limit?

The maximum file size is 1 GB.

Large files are uploaded in chunks, ensuring reliable delivery even on unstable networks. If an upload is interrupted, the system will automatically retry when the network recovers.

What are the ways to upload files?

There are 4 upload methods:

1iTunes File Sharing — Transfer files via Finder / iTunes to the app's sandbox
2Photo Library — Pick a video from your camera roll; audio is extracted automatically
3Files App — Select from iCloud Drive, local storage, or other locations
4WiFi Transfer — Upload from a computer browser on the same local network
📡 WiFi Transfer
How do I use WiFi Transfer?
1Make sure your phone and computer are on the same WiFi network
2Open the WiFi Transfer page in the app — an address will be displayed
3Enter that address in your computer's browser
4Click the upload button on the web page and select your file
5Once the transfer completes, the app will ask you to confirm separation
💡 Keep the app in the foreground during transfer. Leaving the page will stop the service.
WiFi Transfer shows "Unable to get WiFi address"?

Please check the following:

1Confirm your phone is connected to WiFi (not using cellular data)
2In system Settings, make sure the app has "Local Network" permission enabled
3Try disconnecting and reconnecting to WiFi
🎵 Audio Separation
What separation types are available?

10 separation types are supported:

🎵 Vocals & Accompaniment · 🎙️ Noise Removal · 🥁 Drums · 🎸 Bass · 🪕 Acoustic Guitar · 🎸 Electric Guitar · 🎹 Piano · 🎛️ Synthesizer · 🎻 Strings · 🎺 Wind Instruments

Each type uses a specialized AI model to extract the corresponding audio track.

How long does separation take?

Processing time depends on file size and server load:

• Songs under 3 minutes: about 1–3 minutes

• Audio 5–10 minutes long: about 3–8 minutes

You can leave the app during processing. The status will update in History when complete.

Where can I find my results?

Go to the "History" tab in the bottom navigation bar.

Tap a completed task to preview each separated track. You can also share them with other apps.

⚠️ Troubleshooting
What should I do if an upload fails?

Common causes and solutions:

1Unstable network — Switch to a more stable WiFi; the system will auto-retry when connection recovers
2File too large — Make sure the file is under 1 GB
3Unsupported format — Verify the file is a supported audio or video format
4Server busy — Wait a few minutes, then tap Retry in History
💡 Failed tasks remain in History and can be retried at any time.
"No Network Connection" — Can't open the app?

The app requires an internet connection for account initialization. Please verify:

1Your phone is connected to WiFi or cellular data is enabled
2In iOS Settings → Cellular, confirm the app is allowed to use the network
3If using a VPN, try disabling it

Once your network is confirmed, tap the "Retry" button on the screen.

The progress bar seems stuck at a certain percentage?

After the upload finishes, the server begins processing. The progress bar will advance slowly — this is normal.

If it hasn't completed after 10 minutes, the server may be busy. Check the task status in History and tap Retry if needed.
